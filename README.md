# Backend API for Brainflix project - Brainstation 

##Key Features

-NodeJS
-Express.js
-Multer library.
-REST API.

##Installation
In the projeect directory,

Install dependencies with npm.
### `npm install`

Install nodemon
### `npm install --save-dev nodemon`

Start the server
### `npm dev`
or
### `nodemon server.js`
